Video: rossella_ride · v4: continuous rotation-aware landmark expectations
New in v4 — Camera-Aware Expectation (CAE):
Rotation index (0=pure side, 1=front-facing) computed from shoulder width vs calibrated max.
Per-landmark expected visibility interpolated continuously — no discrete bins.
Verdicts: ok (detected as expected) · occluded_ok (not detected, occlusion expected at this angle) · missing (should have been found) · hallucination (detected with high confidence where it shouldn't be visible).
Landmark dots on frames are now colored by verdict.
ok — detected as expected
occluded_ok — missing but expected
missing — should have been found
hallucination — confident but wrong
